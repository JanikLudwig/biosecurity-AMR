    What this is

    A deterministic, auditable pipeline that turns a reconstructed Staphylococcus aureus genome into an earlier antibiotic-response prediction — one of likely to fail / likely to work / no-call per drug — with a calibrated confidence, an evidence category, and a mandatory lab-confirmation notice. No autonomous LLM sits in the decision path.

    M1 · teammate-owned

    Genome Reader

    AMRFinderPlus turns the FASTA into an AMR gene / mutation feature matrix.

    M2 · this work

    Drug-Target Detector

    pyrodigal calls ORFs; pyhmmer proves each drug's molecular target protein is present — so “works” is never claimed from absence alone.

    M3

    Predictor

    One calibrated logistic regression per antibiotic over the AMR features.

    M4 · M5

    Decision & Report

    Fuses model probability + target gate + determinant into the three-way call.

    How “likely to work” is protected

    A drug is only called likely to work when (1) the model's resistance probability is low and (2) the M2 detector has physically found the drug's molecular target in the genome (e.g. gyrA for fluoroquinolones, folA+folP for co-trimoxazole). If the target cannot be proven present, the result is a no-call, never a false “works”.

    Honest generalization

    Near-identical genomes are collapsed by cgMLST (hc10) and whole MLST lineages are held out for testing, so reported performance reflects unseen clonal groups — not memorised near-duplicates.

    Responsibility

    • Defensive by construction — read-only prediction of existing resistance.
    • Calibrated confidence + no-call for weak, conflicting, or out-of-distribution input.
    • Honest explanations — known determinant (i) vs statistical association (ii) vs no signal (iii) are labelled separately.
    • Human oversight — every report must be confirmed by lab testing.
